My Favorite Podcast Apps - Android + Windows

2020-10-06 :: earnest
#podcast  #windows 10 

Jump to: ⭐ Pocket Casts – AntennaPod – Podcast Addict – Grover – Switching

Where I listen⌗

Mostly on Android – my phone or occasionally tablet, followed by my laptop (Windows 10/ Linux).

What’s important to me

Trim silent audio, syncing playback/ listening time and history for at least Android and Windows 10.

⭐ Pocket Casts⌗

This is the app I’m using now.

There are tons of customization/ listening/ player settings.

“Filters” are great to sort out episodes – for example, I have a playlist for any downloaded episodes under half an hour.

This is also a cool feature I don’t think I’ve seen anywhere - grouping episodes.

On the free version, you can sync between the mobile apps Google Play, it’s also available for iOS.

Upgrading to the pro plan – $1.29 CAD / month ($0.99 USD / month or $9.99 USD / year), you get:

  • More themes (by default, there is light, dark, indido, extra dark for free)
    • Additionally: Electricity, classic, radioactivity
    • Additional App Icons
    • Standalone app for Apple Watches
    • 10GB of Cloud Storage (for audio not available as a podcast)
    • Desktop Apps (that sync with your account)
      • Windows 10
      • macOS
      • Web app

The windows and web app work nicely, however there is no download/ offline option. Custom filters also don’t seem to show up on the Windows app.

Syncing works very quickly, and I haven’t had any issues with it yet.

I’ve been using Pocket Casts for a month, and the plus version for 2 weeks. $1.29 a month is quite affordable, and worth it for the quick syncing.

  • Audio
    • ✔ Private feeds
    • ✔ Skip silent audio
    • ✔ Playback speed
    • ✔ Volume boost
  • Interface
    • ✔ Search
    • ✔ Widgets
    • ✔ Sleep timer
    • ✔ Auto downloads

AntennaPod⌗

You may want to use AntennaPod as it is open source and available on both Google Play and F-Droid.

The user interface and player is also pretty customizable.

  • Audio
    • ✔ Private feeds
    • ✔ Skip silent audio
    • ✔ Playback speed
    • ✔ Volume boost
  • Interface
    • ✔ Search
    • ✔ Widgets
    • ✔ Sleep timer
    • ✔ Auto downloads

AntennaPod integrates nicely with gpodder.net.

gpodder.net is a web service to manage your podcast subscriptions via the web. You can synchronize your devices, view status information and discover new interesting podcasts online.

Guess what? It’s also open source! (AGPL-3.0)

You can synchronize with other supported clients. On Windows, I used gPodder; it’s also available for macOS, and Linux/ BSD (GPL-3.0).

On web, there is Cloud Caster, but I haven’t tried it.

However, there are a few issues.

The syncing has to be started manually (on AntennaPod) and is not instant. There is a small delay of a few seconds.

I’ve has a few minor issues as well with the playback times and auto-deletion of played episodes.

Podcast Addict⌗

One of the first apps I used. Podcast Addict is one of the best and powerful podcast players available.

It’s super easy to sort/ organize/ categorize episodes, change the theming and interface, player, etc.

  • Audio
    • ✔ Private feeds
    • ✔ Skip silent audio
    • ✔ Playback speed
    • ❔ Volume boost (volume adjustment)
  • Interface
    • ✔ Search
    • ✔ Widgets
    • ✔ Sleep timer
    • ✔ Auto downloads

Unfortunately for me, there is no syncing feature. However, there is a native backup/ restore feature that keeps all your data safe for the app.

Premium

There are 2 premium options. The first one is ad-free, which is a one-time purchase $9.49 CAD (at the time of this post).

The premium version is $1.29 / month. You get:

  • Ad free
  • Select opening screen / more themes
  • Hide sponsored podcasts

Grover⌗

Used this briefly. Probably the best podcast app for Windows 10 (only – no Android) so far. If you only use Windows 10, this is probably the app for you. Episodes do not auto download, and I don’t think there is a sync option pro version only (via Onedrive), so I haven’t tried it.

  • Audio
    • ✔ Private feeds
    • ❌ Skip silent audio
    • ✔ Playback speed
    • ❌ Volume boost (volume adjustment)
  • Interface
    • ✔ Search
    • Widgets (not on windows, lol)
    • ❌ Sleep timer
    • ❌ Auto downloads

There is a pro version of the app which is currently a one-time payment of $3.89 CAD.

Switching⌗

All of these apps (and others) are pretty easy to switch between. These all offer OPML Import/ Export, bringing in your list of subscribed podcasts. All I’ve had to do is mark all as played.

Thanks for reading! If you have anything to add, please let me know :)
